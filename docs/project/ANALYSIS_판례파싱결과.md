## 🔧 개선 제안

### 1. 판시사항 추출 추가
```python
def extract_panji_items(self, text: str) -> List[str]:
    """판시사항 추출"""
    pattern = r'【?\s*판시사항\s*】?[:\s]*\n?(.+?)(?=\n\n|\n【|$)'
    # [1], [2] 등으로 번호가 매겨진 항목 추출
```

### 2. 참조조문 추출 추가

def extract_judgment_date(self, text: str) -> Optional[str]:
